About The App

 

This app uses Angular, and Spotify's API, and OAuth 2.0 and Material.io. It was born out of a couple of key desires, primarily giving myself some experience in web development; however, the site's premise comes from a long-standing desire to learn about personal musical tastes. I have always wanted to be able to be able to understand my own musical preferences and how they change over time.

 

While this app may not answer all of the burning questions that it could, it has been helpful in my understanding of development in several ways. For one thing, a CSS framework like Bootstrap is a MUST. This website was built without one, and it has been a mess trying to get everything to work correctly. Additionally, I have learned about the limitations of API use. Spotify's API doesn't allow for nearly all of the data collection I was hoping for, such as number of plays for an individual song, but I do think the data visualization aspect of this is pretty cool.
